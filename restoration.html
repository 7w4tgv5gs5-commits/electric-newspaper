About This Archive

How 711 articles were recovered from a dormant 1998 Unix server

The Recovery

From dormant server to public archive

This archive was restored in 2024 from the original Unix tar file extracted from the Solaris server that hosted www.electric.on.net. The tar process had truncated all filenames to eight characters, breaking every image reference in the 711 HTML articles. The restoration process mapped 92% of broken references back to their original files through systematic pattern matching, rewrote 269 absolute URLs to relative paths, fixed cross-browser JavaScript compatibility in 923 files, and built section index pages for every content area.

The article text and body HTML is entirely original and unmodified. The only changes to article files are: image path corrections, navigation link fixes, and the disabling (with preservation in comments) of a Netscape 3-era rollover button script originally written by Dave Sag and Mark King of Virtual Artists in July 1997. The original source code is preserved verbatim in HTML comments in every affected file.

Two image files — news/logosa4.jpg and news/newlogosmall1.gif — were genuinely absent from the tar file and could not be recovered. All other broken references were resolved.

Technical Notes

The Netscape Navigator Experience

The archive is presented inside a simulated Netscape Navigator 4 browser at 800×600 pixels — the browser and screen resolution most Australians used in 1998. The simulation includes working Back and Forward history navigation, a location bar displaying the original http://www.electric.on.net/ URLs, an animated Netscape N logo, and a status bar. It uses only vanilla JavaScript and no external dependencies.

Case Sensitivity

Archive filenames are uppercase (INDEX.HTM, NEWS/, BUTTONS/) because the original Unix server was case-sensitive, as is the Linux server running GitHub Pages. macOS is case-insensitive and will hide path errors locally. All links within the archive use the exact original uppercase filenames.

Internet Service Provider Logos

Internode and Cobweb Internet Services logos are retained at the foot of each article page exactly as they appeared in 1998. Both were sponsors and infrastructure partners of The Electric Newspaper. Simon Hackett and Robyn Taylor founded Internode in Adelaide in 1991; it became one of Australia's leading independent ISPs before being acquired by iiNet in 2012.
